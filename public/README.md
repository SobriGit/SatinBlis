# 🌸 SatinBlis - Premium Bouquet Catalog

SatinBlis adalah platform katalog digital premium untuk bisnis floret/buket yang dibangun dengan fokus pada kecepatan akses dan kemudahan pengelolaan konten.

## 🚀 Fitur Utama
* **Instant Loading:** Menggunakan sistem memori lokal untuk memuat identitas web tanpa delay.
* **Dynamic Admin Dashboard:** Pengelolaan harga, stok, dan nomor WhatsApp secara real-time.
* **Firebase Integration:** Database menggunakan Firestore dan Hosting menggunakan Firebase.
* **Responsive Design:** Tampilan mewah yang optimal di HP maupun Laptop.

## 🛠️ Teknologi yang Digunakan
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+).
* **Backend:** Firebase (Authentication & Firestore).
* **Tools:** Git & Firebase Hosting.
